---
title: Godot Find/Replace Shortcuts
date: "2024-10-30"
excerpt: "Keyboard shortcuts to find and replace in Godot are a bit different."
---

### Find in project (all files)
```js
CTRL + SHIFT + F // Windows
CMD + SHIFT + F // Mac
```


### Find in current file
```js
CTRL + F // Windows
CMD + F // Mac
```

### Replace in current file
 ```js
CTRL + R // Windows
CMD + R // Mac
```
